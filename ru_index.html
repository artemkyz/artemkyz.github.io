Приветствую, гость

Моё имя Артём. Мне нравится создавать веб-приложения и автоматизировать рутинные задачи. Но больше всего мне нравится помогать людям решать проблемы.

Опыт

Состоит из работы на нескольких заводах и в IT-компаниях. В мои обязанности входило проектирование и монтаж слаботочных сетей; сборка ПК для пользователей, установка и настройка приложений; обслуживание оргтехники; администрирование серверов на базе ОС Windows (Active Directory, Domain Controller, IIS); администрирование серверов на базе ОС Linux (прокси-серверы: Apache и Nginx, сервер резервного копирования, развертывание сервисов приложений). Работа в поддержке на 1-й, 2-й и 3-й линиях; тестирование программного обеспечения и исправление ошибок.

Инструменты

Создание веб-приложений состоит из нескольких этапов. Бэкэнд - пишется на Python с использованием фреймворка Flask. Интерфейс создается на HTML, CSS, javaScript. Созданное приложение запускается на сервере с операционной системой Linux и веб-сервером Nginx или Apache, в зависимости от конкретной задачи.
